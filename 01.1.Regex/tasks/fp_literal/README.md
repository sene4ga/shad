## Числовой литерал

### Введение

См. аналогичный раздел в задании про шестнадцатиричный числовой литерал.


### Условие

Исходим из того, что у нас есть функция

```python
def extract_token(s, regexp, f_repr):
    if m := re.match(regexp, s):
        return f_repr(m), s[m.span()[1]:]
    else:
        return None, s
```

которая принимает:

- строку s

- регулярное выражение regexp

- функцию f_repr

Регулярное выражение должно привести в возвращению непустного Match-объекта, если

- строка s начинается с вещественного литерала (в классической или "научной" нотации), которому может
предшествовать пробельная последовательность

- за последней цифрой литерала следует пробел, конец строки или знак арифметической операции
 
Непустой Match-объект, должен возвращать span(), заканчивающийся ровно на последнем символе литерала.

Решение должно состоять из:

- регулярного выражения, сохраненного в переменной RE_FP_LITERAL

- функции `f_repr_fp_literal`, принимающей непустой Match-объект и возвращающей объект типа
FPLiteral со следующими полями:

-- integral: строковое предстравление целой части числа

-- fractional: строковое представление дробной части числа

-- exp: строковое представление экспоненты (None, если она отсутствует совсем)


### Примеры

'  1.234+v' - должен получиться непустой Match-объект и `extract_token` должна вернуть
пару (FPLiteral(integral='1', fractional='234', exp=None), '+v')

'  .12340e+12' - должен получиться непустой Match-объект и `extract_token` должна вернуть
пару (FPLiteral(integral='', fractional='12340', exp='+12'), '')

'  01234x' - extract_token` должна вернуть пару (None, '  01234x')
