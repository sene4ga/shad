## Числовой литерал


### Условие

Исходим из того, что у нас есть функция

```python
def sub_all(s, r, subst):
    return re.sub(r, subst, s)
```

которая принимает:

- строку s

- регулярное выражение r

- подстановочное выражение subst

Регулярное и подстановочные выражения должно быть написано таким образом, чтобы приведенный вызов заменил любую из
дат формата 'dd.mm.yyyy', на дату формата 'mm/dd/yyyy'.

Кроме того:

- месяц может состоять из одной цифры
- день может состоять из одной цифры
- год может состоять из двух цифр

Если нарушено любое из следующих условий, то последовательность символов датой не считается и за другой формат не заменяется:

- день является числом в диапазоне [1, 31]

- месяц является числом в диапазоне [1, 12] 

- если год представлен 4 цифрами, то это число в диапазоне [1000, 2999]

Чтобы последовательность символов считалась датой и подлежажала замене на другой формат, надо, чтобы:

- непосредственно перед ней шел любой из символов '(', ' ', ',', ';' или начало строки 

- непосредственно перед ней шел любой из символов ')', ' ', ',', ';' или конец строки 


При замене на другой формат число цифр в дне/месяце/годе должно остаться без изменений (например, 01 остается 01).


### Примеры

'12.08.1234 12.00.1234' - должно превратиться в '08/12/1234 12.00.1234' 

'12.08.1234 12.1.1234' - должно превратиться в '08/12/1234 1/12/1234'
